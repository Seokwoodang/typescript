# 타입 명시해주기 vs 자동으로 타입스크립트에게 맡겨버리기

타입을 직접 명시하는 방법과 초기화 부분에서 타입스크립트에게 맡기는 방법이 있다.

## 타입을 직접 명시해주는 방법
```
let a = 'hello'
let b = true
let c = [1,2,3]
```

## 모든걸 맡기는 방법
```
let a : string = 'hello';
let b : boolean = true;
let c : number[] = [1,2,3];
```

코드의 가독성과 정확성을 위해서 웬만하면 직접 명시해주는 방법을 사용하도록 하자!!
